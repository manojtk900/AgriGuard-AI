# 🌾 AgriGuard AI

### AI-Based Data-Driven Predictive Farming System with Explainable AI and Financial Insights

---

## 🚀 Overview

AgriGuard AI is an intelligent web-based agriculture decision support system designed to assist farmers in making data-driven, predictive, and financially informed decisions.

The system integrates machine learning, explainable AI, and financial analysis to improve crop selection, disease detection, and overall farm productivity.

---

## 💡 Key Features

### 🌱 Crop Prediction

* Predicts suitable crops based on:

  * Temperature
  * Humidity
  * Soil pH
  * Rainfall
* Provides risk level (Low / Medium / High)

---

### ⚠️ Risk Analysis

* Identifies potential crop risks
* Helps farmers take preventive actions

---

### 🧠 Explainable AI

* Provides reasoning behind predictions
* Improves transparency and trust

---

### 📸 Disease Detection

* Upload plant leaf image
* Detect diseases like:

  * Healthy
  * Early Blight
  * Late Blight
* Shows:

  * Confidence score
  * Severity level
  * Pathology details
  * Treatment recommendations

---

### 💰 Financial Insights

* Calculates:

  * Cost
  * Expected yield
  * Estimated profit
* Includes data visualization (graphs)

---

### 🧊 3D Plant Visualization

* Interactive 3D plant model
* Helps visualize crop structure

---

### 🏛️ Government Subsidies

* Information on schemes:

  * PM-KISAN
  * PMFBY
  * PMKSY
* Includes benefits and eligibility

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Bootstrap
* **Backend:** Flask (Python)
* **Visualization:** Chart.js
* **3D Graphics:** Three.js
* **Machine Learning:** Simulated models (Random Forest, CNN)

1. Install dependencies

```

```

2. Run application

```
python app.py
```

3. Open browser

```
http://localhost:5000
```
